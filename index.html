OpenFile Viewerとは

OpenFile Viewerは、PSD、PDF、画像、テキストファイルなどを ブラウザ上で簡単に開くことができる無料オンラインビューアです。

Photoshopなどの専用ソフトがなくても、PSDファイルのレイヤー確認、 単体レイヤー表示、レイヤー合成表示、PNG書き出しが可能です。

デザインデータの確認、納品ファイルのチェック、 簡易プレビューなどに利用できます。

主な機能

  • PSDレイヤー表示
  • レイヤー単体表示
  • レイヤー合成表示
  • PNGエクスポート
  • PDFビューア
  • 画像プレビュー
  • テキスト表示

対応ファイル形式

PSD / PDF / PNG / JPG / JPEG / GIF / WEBP / SVG / TXT / JSON / CSV / MD

このサイトはすべてブラウザ上で動作するため、 アップロードされたファイルはサーバーに保存されません。
